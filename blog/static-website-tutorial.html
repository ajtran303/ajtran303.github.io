Tutorial: Build a Simple Static Website

Building a simple static website using HTML and CSS is a great way to get started with web development. A static website is one that displays the same content to all users, and does not require any server-side processing or dynamic content. In this tutorial, we'll walk through the steps of creating a basic static website using only HTML and CSS.

  1. Create a new folder on your computer to store your website files. This will be your project directory.

  2. Create a new file in your project directory called "index.html". This will be the main page of your website.

  3. Open "index.html" in a text editor, and add the following HTML code to create a basic structure for your website: 

    
<!DOCTYPE html>
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
  </body>
</html>

  4. Within the <body> element, add some content for your website. You can use HTML tags to structure your content, such as headings, paragraphs, lists, and images. For example: 

    
<body>
  <h1>Welcome to My Website</h1>
  <p>This is a simple static website built using HTML and CSS.</p>
  <ul>
    <li>About</li>
    <li>Contact</li>
  </ul>
</body>

  5. To add styling to your website, create a new file in your project directory called "style.css". This will be your stylesheet file.

  6. Open "style.css" in a text editor, and add some basic styles to your website. For example: 

    
body {
  font-family: Arial, sans-serif;
  color: #333;
}

h1 {
  color: #00b8d4;
}

ul {
  list-style: none;
}

li {
  display: inline-block;
  margin-right: 20px;
}

  7. To link your stylesheet to your HTML file, add a <link> element to the <head> element of your HTML file: 

    
<head>
  <title>My Website</title>
  <link rel="stylesheet" href="style.css">
</head>

  8. Save your changes and open "index.html" in a web browser to view your website. You should see your content displayed with the styles you defined in your stylesheet.

That's it! You've created a simple static website using HTML and CSS. As you continue to learn more about web development, you can add more features and functionality to your website using other technologies, such as JavaScript and server-side programming.
